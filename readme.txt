Bonjour,
Vous trouverez a la racine du projet parent, les projets suivant:

- acteur-somme
- acteur-supervisor

Le projet acteur-somme contient les sources de l'exemple sur la liste d'entiers dont les elements vont etre multipies par 2.

Le projet acteur-supervisor contient les sources de l'exemple sur la supervision. A noter que cette partie sera exposee 
dans le magazine "Programmez!" de fin janvier.

Pour executer les programmes vous pouvrez lancer la class contenant la méthode main ou vous mettre a la racine de chaque projet
et de taper la commande: mvn clean test .

Pour compiler et tester les sources de ces projets vous devez disposer des composants suivants:

- jdk 1.7
- Maven 3
